# Log — Running History

*The assistant adds a short entry here every time it changes your memory, so you can always see what happened and when. Newest at the bottom.*

---

*Format the assistant follows for each entry:*

```
## [YYYY-MM-DD] action | Title
- Topic: {work | learning | personal}
- Source: {raw file, if any}
- Pages created: {list}
- Pages updated: {list}
```

*Actions: fed-in, asked, tidied, sweep.*
